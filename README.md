## コマンド
| コマンド名           | 説明       | 権限    |
|-----------------|----------|-------|
| /cpsgame show   | スコアの確認   | ----- |
| /cpsgame summon | ターゲットを召喚 | OP    |
| /cpsgame delete | ターゲットを削除 | OP    |

## configファイル
| key名                 | 説明            | デフォルト値 |
|----------------------|---------------| ------ |
| limit_time           | 制限時間          | 10 |
| is_write_scoreboard  | スコアボードに保存するか  | false |
|scoreboard_name| 保存するスコアボードの名前 | "" |
| scoreboard_display_name | 保存するスコアボードのディスプレイ名前 |  |
